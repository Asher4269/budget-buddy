# budget-buddy

Budget Buddy: Free Companion for Personal Finance

By: Asher Ruiz

## Description

-

- “Totals” and “Income Calculator” don’t update with crud functionality. Ex. Stuart and Trevor tried to edit or delete items, and the categorical total would not change until they hit load budget again.
- “Type” Confusion: people did not know the difference between budget, expense, and savings. This makes sense, as this jargon is not intuitive. I am going to add some explanations to make it easier.
- Unexpectedly, despite my initial input prefix being “ex. 400.00” for amount, people still tried to put “$400.00”.
- Some did not like how when they refresh the page, all of their inputs disappear.
